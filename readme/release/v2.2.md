# 最新版本：EasyPoi 2.4

## 更新内容
1 在v2.3版本上优化设置项布局

2 解决部分采集过程中出现的bug

3 优化异常处理机制，部分区域采集失败时不会中断采集流程

## 注意事项

**1 安装路径中不能含有中文，非常重要！**

地图打点在程序实现层面引入了新的模块，不支持中文路径，如果安装路径中含有中文，在使用地图打点功能时会闪退。

2 从EasyPoi 2.2版本开始，会安装到新的目录，不会覆盖安装原有的版本，建议先卸载再安装
